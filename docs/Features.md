# Features

This document lists the main features of DinoSweeper.

---

## 🎮 Gameplay

- Classic Minesweeper mechanics: reveal cells, flag mines.
- Dynamic board rendering with responsive UI.
- Mines counter and timer display.
- Hint system with difficulty-based limits.
- Victory/lose dialog with themed images.

---

## ⚙️ Difficulty Levels

- **Beginner**: 9x9 grid, 10 mines, unlimited hints.
- **Intermediate**: 16x16 grid, 40 mines, 3 hints.
- **Expert**: 30x16 grid, 99 mines, no hints.

---

## ⏱️ Timer Modes

- **Normal Mode**: Count up from zero.
- **Countdown Mode**: Predefined limits (300s, 180s, 60s).

---

## 🖥️ Settings

- Difficulty selection.
- Mode selection (Normal / Countdown).
- Countdown level options.

---

## 🎨 UI & Assets

- Centralized asset management (`Assets.cs`).
- Interactive icons for hints and mines.
- Result dialogs with victory/lose images.
