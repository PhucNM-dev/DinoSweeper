# Installation Guide

This document explains how to set up and run the DinoSweeper project locally.

---

## 📋 Prerequisites

Before installing, make sure you have:

- **Windows 10/11**  
- **Visual Studio 2022** (Community edition or higher)  
- **.NET Desktop Development workload** installed in Visual Studio  
- **Git** (optional, for cloning the repository)

---

## 🚀 Steps to Install

1. **Clone the repository**
   ```bash
   git clone https://github.com/PhucNM-dev/DinoSweeper.git

2. **Open the solution**
   - Navigate to the project folder.
   - Double-click `MineSweeper.sln` to open in Visual Studio.

3. **Build the project**
   - In Visual Studio, press `Ctrl+Shift+B` or go to **Build → Build Solution**.

4. **Run the project**
   - Press `F5` or click **Start** to launch DinoSweeper.

---

## ⚙️ Configuration

- **Difficulty & Modes**: Accessible via the Settings window in the application.  
- **Countdown Timer**: Select from predefined options (300s, 180s, 60s).  
- **Assets**: Images and icons are stored in the `Assets/` folder.  

---

## 🛠 Troubleshooting

- **Build fails** → Ensure the **.NET Desktop Development workload** is installed in Visual Studio.  
- **Missing assets** → Verify the `Assets/` folder is included in the project.  
- **Git issues** → Check that you have the correct repository URL.  

---

## ✅ Verification

After installation, you should be able to:
- Launch the game from Visual Studio.  
- Select difficulty and mode.  
- Play Minesweeper with timer and hint features.
